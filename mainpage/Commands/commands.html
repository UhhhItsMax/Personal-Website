Der EhrenBot

Commands:

Moderation
  • !ban [User ID]   - Dieser Command bannt den Member.
  • !kick [User ID]   - Dieser Command kickt den Member.
  • !mute [User ID]   - Dieser Command muted den Member.
  • !unmute [User ID]   - Dieser Command nimmt den User.
Rollen
  • !add [Rollen name, Rollen name, ...]   - Die genannte Rolle wird hinzugefügt, es können mehrere Rollen gleichzeitig hinzugefügt werden.
  • !del [Rollen name, Rollen name, ...]   - Die genannte Rolle wird entfernt, sofern der Member die Rolle hat, es können mehrere Rollen gleichzeitig entfernt werden.
Fun

  • !jesus   - Schickt ein Bild von Jesus. AMEN!
  • !rps [rock, paper, scissors]   - Dieser Command spielt Schere Stein Papier mit dir.
